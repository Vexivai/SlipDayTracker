# SlipDayTool 

This tool helps the TAs track the active students they have in their roster. 
I have a Tutorial.pdf to reference in the files. 

I took the CSV download approach instead of trying to go through the SSO 
sign in approach. Otherwise I would have had to have each user set up their 
own sign in privileges, and scrape and create each detail manually. By the 
time I would have gotten any closer to that goal, I would have graduated..
